## Tech Stack

- **Runtime**: Node.js 18.x
- **Framework**: Express.js 4.x
- **Database**: MongoDB 6.x
- **ORM**: Mongoose
- **Language**: TypeScript 5.x
- **Authentication**: JWT

## Getting Started

### Prerequisites

- Node.js 18.x
- MongoDB 6.x
- npm 9.x or yarn 1.x

### Installation

1. install nodemodules:
    ```bash
    npm i

2. run in dev mode:
    ```bash
    npm run start:dev

3. run in prod mode:
    ```bash
    npm run start:prod
